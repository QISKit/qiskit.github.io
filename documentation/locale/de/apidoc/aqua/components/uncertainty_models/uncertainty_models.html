abstract build(qc, q, q_ancillas=None)¶

Adds corresponding sub-circuit to given circuit

Parameter

  • qc – quantum circuit

  • q – list of qubits (has to be same length as self._num_qubits)

  • q_ancillas – list of ancilla qubits (or None if none needed)

  • params – parameters for circuit

class UnivariateDistribution(num_target_qubits, probabilities, low=0, high=1)[Quellcode]¶

This module contains the definition of a base class for univariate distributions. (Interface for discrete bounded uncertainty models assuming an equidistant grid)

Abstract univariate distribution class :param num_target_qubits: number of qubits it acts on :type num_target_qubits: int :param probabilities: probabilities for different states :type probabilities: array or list :param low: lower bound, i.e., the value corresponding to |0…0> (assuming an equidistant grid) :type low: float :param high: upper bound, i.e., the value corresponding to |1…1> (assuming an equidistant grid) :type high: float
