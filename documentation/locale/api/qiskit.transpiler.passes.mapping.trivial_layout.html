qiskit.transpiler.passes.mapping.trivial_layout module¶

A pass for choosing a Layout of a circuit onto a Coupling graph, using a simple round-robin order.

This pass associates a physical qubit (int) to each virtual qubit of the circuit (tuple(QuantumRegister, int)) in increasing order.

class TrivialLayout(coupling_map)[ソース]¶

ベースクラス: qiskit.transpiler.basepasses.AnalysisPass

Chooses a Layout by assigning n circuit qubits to device qubits 0, .., n-1.

Does not assume any ancilla.

run(dag)[ソース]¶

Pick a layout by assigning n circuit qubits to device qubits 0, .., n-1.

パラメータ:dag (DAGCircuit) -- DAG to find layout for.
Raises:TranspilerError -- if dag wider than self.coupling_map
