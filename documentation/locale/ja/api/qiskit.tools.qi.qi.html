qiskit.tools.qi.qi module¶

A collection of useful quantum information functions.

Currently this file is very sparse. More functions will be added over time.

choi_to_rauli(choi, order=1)[ソース]¶

Convert a Choi-matrix to a Pauli-basis superoperator.

Note that this function assumes that the Choi-matrix is defined in the standard column-stacking convention and is normalized to have trace 1. For a channel E this is defined as: choi = (I otimes E)(bell_state).

The resulting 'rauli' R acts on input states as |rho_out>_p = R.|rho_in>_p where |rho> = vectorize(rho, method='pauli') for order=1 and |rho> = vectorize(rho, method='pauli_weights') for order=0.

パラメータ:
  • choi (matrix) -- the input Choi-matrix.
  • order (int) --

    ordering of the Pauli group vector. order=1 (default) is standard lexicographic ordering.

    Eg: [II, IX, IY, IZ, XI, XX, XY,...]
    order=0 is ordered by weights.
    Eg. [II, IX, IY, IZ, XI, XY, XZ, XX, XY,...]
戻り値:

A superoperator in the Pauli basis.
戻り値の型:

np.array

devectorize(vectorized_mat, method='col')[ソース]¶

Devectorize a vectorized square matrix.

パラメータ:
  • vectorized_mat (ndarray) -- a vectorized density matrix.
  • method (str) --

    the method of devectorization. Allowed values are - 'col' (default): flattens to column-major vector. - 'row': flattens to row-major vector. - 'pauli': flattens in the n-qubit Pauli basis. - 'pauli-weights': flattens in the n-qubit Pauli basis ordered by

    weight.
戻り値:

the resulting matrix.
戻り値の型:

ndarray
Raises:

Exception -- if input state is not a n-qubit state
entanglement_of_formation(state, d0, d1=None)[ソース]¶

Compute the entanglement of formation of quantum state.

The input quantum state must be either a bipartite state vector, or a 2-qubit density matrix.

パラメータ:
  • state (array_like) -- (N) array_like or (4,4) array_like, a bipartite quantum state.
  • d0 (int) -- the dimension of the first subsystem.
  • d1 (int or None) -- the dimension of the second subsystem.
戻り値:

The entanglement of formation.
戻り値の型:

float

outer(vector1, vector2=None)[ソース]¶

パラメータ:
  • vector1 (ndarray) -- the first vector.
  • vector2 (ndarray) -- the (optional) second vector.
戻り値:

戻り値の型:

np.array
partial_trace(state, trace_systems, dimensions=None, reverse=True)[ソース]¶

Partial trace over subsystems of multi-partite matrix.

Note that subsystems are ordered as rho012 = rho0(x)rho1(x)rho2.

パラメータ:
  • state (matrix_like) -- a matrix NxN
  • trace_systems (list(int)) -- a list of subsystems (starting from 0) to trace over.
  • dimensions (list(int)) -- a list of the dimensions of the subsystems. If this is not set it will assume all subsystems are qubits.
  • reverse (bool) -- ordering of systems in operator. If True system-0 is the right most system in tensor product. If False system-0 is the left most system in tensor product.
戻り値:

A density matrix with the appropriate subsystems traced

over.
戻り値の型:

matrix_like
Raises:

Exception -- if input is not a multi-qubit state.
qft(circ, q, n)[ソース]¶

n-qubit QFT on q in circ.

shannon_entropy(pvec, base=2)[ソース]¶

Compute the Shannon entropy of a probability vector.

The shannon entropy of a probability vector pv is defined as $H(pv) = - sum_j pv[j] log_b (pv[j])$ where $0 log_b 0 = 0$.

パラメータ:
  • pvec (array_like) -- a probability vector.
  • base (int) -- the base of the logarithm
戻り値:

The Shannon entropy H(pvec).
戻り値の型:

float

    weight.
戻り値:

the resulting vector.
戻り値の型:

ndarray
Raises:

Exception -- if input state is not a n-qubit state
