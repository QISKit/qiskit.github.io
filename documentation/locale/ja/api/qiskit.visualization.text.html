qiskit.visualization.text module¶

A module for drawing circuits in ascii art or some other text representation

class Barrier(label='')[ソース]¶

ベースクラス: qiskit.visualization.text.DirectOnQuWire

Draws a barrier.

top:  ░     ░
mid: ─░─ ───░───
bot:  ░     ░
class BoxOnClWire(label='', top_connect='─', bot_connect='─')[ソース]¶

ベースクラス: qiskit.visualization.text.DrawElement

Draws a box on the classical wire.

top: ┌───┐   ┌───┐
mid: ╡ A ╞ ══╡ A ╞══
bot: └───┘   └───┘
class BoxOnClWireBot(label, input_length, bot_connect='─', wire_label='', **_)[ソース]¶

ベースクラス: qiskit.visualization.text.MultiBox, qiskit.visualization.text.BoxOnClWire

Draws the bottom part of a conditional box that affects more than one classical wire

class BoxOnClWireMid(label, input_length, order, wire_label='')[ソース]¶

ベースクラス: qiskit.visualization.text.BoxOnWireMid, qiskit.visualization.text.BoxOnClWire

Draws the middle part of a conditional box that affects more than one classical wire

class BoxOnClWireTop(label='', top_connect=None, wire_label='')[ソース]¶

ベースクラス: qiskit.visualization.text.MultiBox, qiskit.visualization.text.BoxOnClWire

Draws the top part of a conditional box that affects more than one classical wire

class BoxOnQuWire(label='', top_connect='─', conditional=False)[ソース]¶

ベースクラス: qiskit.visualization.text.DrawElement

Draws a box on the quantum wire.

top: ┌───┐   ┌───┐
mid: ┤ A ├ ──┤ A ├──
bot: └───┘   └───┘
class BoxOnQuWireBot(label, input_length, bot_connect='─', wire_label='', conditional=False)[ソース]¶

ベースクラス: qiskit.visualization.text.MultiBox, qiskit.visualization.text.BoxOnQuWire

Draws the bottom part of a box that affects more than one quantum wire

class BoxOnQuWireMid(label, input_length, order, wire_label='')[ソース]¶

ベースクラス: qiskit.visualization.text.BoxOnWireMid, qiskit.visualization.text.BoxOnQuWire

Draws the middle part of a box that affects more than one quantum wire

class BoxOnQuWireTop(label='', top_connect=None, wire_label='')[ソース]¶

ベースクラス: qiskit.visualization.text.MultiBox, qiskit.visualization.text.BoxOnQuWire

Draws the top part of a box that affects more than one quantum wire

class BoxOnWireMid(label, input_length, order, wire_label='')[ソース]¶

ベースクラス: qiskit.visualization.text.MultiBox

A generic middle box

class BreakWire(arrow_char)[ソース]¶

ベースクラス: qiskit.visualization.text.DrawElement

This element is used to break the drawing in several pages.

static fillup_layer(layer_length, arrow_char)[ソース]¶

Creates a layer with BreakWire elements. :param layer_length: The length of the layer to create :type layer_length: int :param arrow_char: The char used to create the BreakWire element. :type arrow_char: char

戻り値:The new layer.
戻り値の型:list
class Bullet(top_connect='', bot_connect='', conditional=False)[ソース]¶

ベースクラス: qiskit.visualization.text.DirectOnQuWire

Draws a bullet (usually with a connector). E.g. the top part of a CX gate.

top:
mid: ─■─  ───■───
bot:  │      │
class DirectOnQuWire(label='')[ソース]¶

ベースクラス: qiskit.visualization.text.DrawElement

Element to the wire (without the box).

class DrawElement(label=None)[ソース]¶

ベースクラス: object

An element is an instruction or an operation that need to be drawn.

bot¶

Constructs the bottom line of the element

connect(wire_char, where, label=None)[ソース]¶

Connects boxes and elements using wire_char and setting proper connectors. :param wire_char: For example '║' or '│'. :type wire_char: char :param where: Where the connector should be set. :type where: list["top", "bot"] :param label: Some connectors have a label (see cu1, for example). :type label: string

length¶

Returns the length of the element, including the box around.

mid¶

Constructs the middle line of the element

top¶

Constructs the top line of the element

width¶

Returns the width of the label, including padding

class EmptyWire(wire)[ソース]¶

ベースクラス: qiskit.visualization.text.DrawElement

This element is just the wire, with no instructions nor operations.

static fillup_layer(layer, first_clbit)[ソース]¶

Given a layer, replace the Nones in it with EmptyWire elements. :param layer: The layer that contains Nones. :type layer: list :param first_clbit: The first wire that is classic. :type first_clbit: int

戻り値:The new layer, with no Nones.
戻り値の型:list
class Ex(bot_connect=' ', top_connect=' ', conditional=False)[ソース]¶

ベースクラス: qiskit.visualization.text.DirectOnQuWire

Draws an X (usually with a connector). E.g. the top part of a swap gate.

top:
mid: ─X─ ───X───
bot:  │     │
class InputWire(label)[ソース]¶

ベースクラス: qiskit.visualization.text.DrawElement

This element is the label and the initial value of a wire.

static fillup_layer(names)[ソース]¶

Creates a layer with InputWire elements. :param names: List of names for the wires. :type names: list

戻り値:The new layer
戻り値の型:list
class Layer(qregs, cregs)[ソース]¶

ベースクラス: object

A layer is the "column" of the circuit.

connect_with(wire_char)[ソース]¶

Connects the elements in the layer using wire_char. :param wire_char: For example '║' or '│'. :type wire_char: char

full_layer¶

Returns the composition of qubits and classic wires. :returns: self.qubit_layer + self.clbit_layer :rtype: String

set_cl_multibox(creg, label, top_connect='┴')[ソース]¶

Sets the multi clbit box. :param creg: The affected classical register. :type creg: string :param label: The label for the multi clbit box. :type label: string :param top_connect: The char to connect the box on the top. :type top_connect: char

set_clbit(clbit, element)[ソース]¶

Sets the clbit to the element :param clbit: Element of self.cregs. :type clbit: cbit :param element: Element to set in the clbit :type element: DrawElement

set_qu_multibox(bits, label, conditional=False)[ソース]¶

Sets the multi qubit box. :param bits: A list of affected bits. :type bits: list[int] :param label: The label for the multi qubit box. :type label: string :param conditional: If the box has a conditional :type conditional: bool

set_qubit(qubit, element)[ソース]¶

Sets the qubit to the element :param qubit: Element of self.qregs. :type qubit: qbit :param element: Element to set in the qubit :type element: DrawElement

class MeasureFrom[ソース]¶

ベースクラス: qiskit.visualization.text.BoxOnQuWire
