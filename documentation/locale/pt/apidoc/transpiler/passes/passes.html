qiskit.transpiler.passes¶

Module containing transpiler pass.

Classes¶

ApplyLayout() Transforms a DAGCircuit with virtual qubits into a DAGCircuit with physical qubits by applying the Layout given in property_set.
BarrierBeforeFinalMeasurements() Adds a barrier before final measurements.
BasicSwap(coupling_map) Maps (with minimum effort) a DAGCircuit onto a coupling_map adding swap gates.
CXCancellation() Cancel back-to-back “cx” gates in dag.
CXDirection(coupling_map) Rearranges the direction of the cx nodes to make the circuit compatible with the directed coupling map.
CheckCXDirection(coupling_map) Checks if the CNOTs in the DAG circuit follow the right direction with respect to the coupling map.
CheckMap(coupling_map) Checks if a DAGCircuit is mapped to coupling_map, setting is_swap_mapped in the property set as True if mapped.
Collect2qBlocks() Pass to collect sequences of uninterrupted gates acting on 2 qubits.
CommutationAnalysis() An analysis pass to find commutation relations between DAG nodes.
CommutativeCancellation() Transformation pass that cancels the redundant (self-adjoint) gates through commutation relations
ConsolidateBlocks([kak_basis_gate, …]) Pass to consolidate sequences of uninterrupted gates acting on the same qubits into a Unitary node, to be resynthesized later, to a potentially more optimal subcircuit.
CountOps() An analysis pass for counting operations in a DAG circuit.
CountOpsLongestPath() An analysis pass for counting operations on the longest path in a DAGcircuit.
DAGFixedPoint() A dummy analysis pass that checks if the DAG a fixed point.
DAGLongestPath() An analysis pass that returns the longest path in a DAGcircuit as a list of DAGNodes.
Decompose([gate]) Expand a gate in a circuit using its decomposition rules.
DenseLayout(coupling_map) Chooses a Layout by finding the most connected subset of qubits.
Depth() An analysis pass for calculating the depth of a DAG circuit.
EnlargeWithAncilla([layout]) Extends the DAG circuit with virtual qubits (ancilla) that are specified in the layout, but not present in the circuit.
FixedPoint(property_to_check) A dummy analysis pass that checks if a property reached a fixed point.
FullAncillaAllocation(coupling_map[, layout]) Allocates all idle nodes from the coupling map as ancilla on the layout.
LookaheadSwap(coupling_map) Map input circuit onto a backend topology via insertion of SWAPs.
MergeAdjacentBarriers() Returns a circuit with any adjacent barriers merged together
NoiseAdaptiveLayout(backend_prop) Chooses a noise-adaptive Layout based on current calibration data for the backend.
NumTensorFactors() An analysis pass for calculating the number of tensor factors of a DAG circuit.
Optimize1qGates() Simplify runs of single qubit gates in the [«u1», «u2», «u3», «cx», «id»] basis.
OptimizeSwapBeforeMeasure() Remove the swaps followed by measurement (and adapt the measurement)
RemoveDiagonalGatesBeforeMeasure() Remove diagonal gates (like RZ, T, Z, etc) before a measurement.
RemoveResetInZeroState() Remove reset gate when the qubit is in zero state
ResourceEstimation() Requires Depth(), Width(), Size(), CountOps(), NumTensorFactors(), and NumQubits.
SetLayout(layout) Sets property_set[“layout”] to layout.
Size() An analysis pass for calculating the size of a DAG circuit.
StochasticSwap(coupling_map[, trials, seed]) Maps a DAGCircuit onto a coupling_map adding swap gates.
TrivialLayout(coupling_map) Chooses a Layout by assigning n circuit qubits to device qubits 0, .., n-1.
Unroll3qOrMore() Recursively expands 3+ qubit gates until the circuit only contains 1 qubit and 2qubit gates.
Unroller(basis) Unroll (expand) non-basis, non-opaque instructions recursively to a desired basis, using decomposition rules defined for each instruction.
Width() An analysis pass for calculating the width of a DAG circuit.
