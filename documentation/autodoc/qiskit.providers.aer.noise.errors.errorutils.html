kraus2instructions(kraus_ops, standard_gates, atol=1e-08)[source]¶

Convert a list of Kraus matrices into qobj circuits.

If any Kraus operators are a unitary matrix they will be converted into unitary qobj instructions. Identity unitary matrices will also be converted into identity qobj instructions.

Parameters:
  • kraus_ops (list[matrix]) – A list of Kraus matrices for a CPTP map.
  • standard_gates (bool) – Check if the matrix instruction is a standard instruction (default: True).
  • atol (double) – Threshold for testing if probabilities are zero.
Returns:

A list of pairs (p, circuit) where circuit is a list of qobj instructions, and p is the probability of that circuit for the given error.
Return type:

list
Raises:

NoiseError – If the input Kraus channel is not CPTP.

single_qubit_clifford_gates(j)[source]¶

Return a QASM gate names for a single qubit clifford.

The labels are returned in a basis set consisting of (‘id’, ‘s’, ‘sdg’, ‘z’, ‘h’, x’, ‘y’) gates decomposed to use the minimum number of X-90 pulses in a (u1, u2, u3) decomposition.

Parameters:j (int) – Clifford index 0, …, 23.
Returns:The tuple of basis gates.
Return type:tuple(str)
Raises:NoiseError – If index is out of range [0, 23].
single_qubit_clifford_instructions(index, qubit=0)[source]¶

Return a list of qobj instructions for a single qubit cliffords.

The instructions are returned in a basis set consisting of (‘id’, ‘s’, ‘sdg’, ‘z’, ‘h’, x’, ‘y’) gates decomposed to use the minimum number of X-90 pulses in a (u1, u2, u3) decomposition.

Parameters:
  • index (int) – Clifford index 0, …, 23.
  • qubit (int) – the qubit to apply the Clifford to.
Returns:

The list of instructions.
Return type:

list(dict)
Raises:

NoiseError – If index is out of range [0, 23] or qubit invalid.
