Aqua TutorialsÂ¶

Qiskit Aqua Tutorials is a set of code examples, Jupyter Notebooks and sample input files that enable experimenting with Aqua: A Library of Quantum Algorithms, Aqua Chemistry, Aqua Artificial Intelligence and Aqua Optimization. The programs included in Aqua Tutorials demonstrate how to use Aqua and its domain-specific applications programmatically, and also how to extend Aqua with new components. This feature is particularly useful to quantum researchers who are interested in Contributing to Aqua with new algorithmic components.

The Aqua Tutorials repository is supposed to be installed via the git clone command. This will allow the user to place its code in a convenient location in the file system, where it will be easier to browse the examples and execute the Jupyter Notebooks provided by Aqua Tutorials.

A Jupyter Notebook nbviewer allows for a quick and intuitive way to browse through the various Jupyter Notebooks offered by Aqua Tutorials.
