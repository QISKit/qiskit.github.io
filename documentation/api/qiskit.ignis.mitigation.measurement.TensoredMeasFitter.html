TensoredMeasFitter¶

class TensoredMeasFitter(results, mit_pattern, substate_labels_list=None, circlabel='')[source]¶

Bases: object

Measurement correction fitter for a tensored calibration

Initialize a measurement calibration matrix from the results of running the circuits returned by measurement_calibration_circuits

Parameters

  • results – the results of running the measurement calibration

  • If this is None the user will set calibration matrices (circuits.) –

  • later –

  • mit_pattern (list of lists of integers) – qubits to perform the

  • correction on, divided to groups according to tensors (measurement) –

  • substate_labels_list (list of lists of strings) – for each

  • matrix, the labels of its rows and columns. (calibration) –

  • None then the labels are ordered lexicographically (If) –

readout_fidelity(cal_index=0, label_list=None)[source]¶

Based on the results output the readout fidelity, which is the average of the diagonal entries in the calibration matrices

Parameters

  • cal_index – readout fidelity of which sub cal?

  • label_list (list of lists on states) –

  • the average fidelity over of the groups of states. (Returns) –

  • None then each state used in the construction of the (If) –

  • matrices forms a group of size 1 (calibration) –

Returns
