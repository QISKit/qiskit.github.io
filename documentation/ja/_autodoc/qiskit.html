qiskit package¶

Main QISKit public functionality.

Submodules¶

backends Utilities for using backends.
dagcircuit Module for DAG Circuits.
extensions Extensions to quantum circuits.
mapper Utils for mapping.
qasm Tools for QASM.
transpiler Utils for transpiler.
unroll Unroll QASM and different backends.
wrapper Helper module for simplified QISKit usage.

Classes¶

ClassicalRegister Implement a classical register.
CompositeGate Composite gate, a sequence of unitary gates.
Gate Unitary gate.
Instruction Generic quantum computer instruction.
InstructionSet Instruction collection.
Measure Quantum measurement in the computational basis.
QuantumCircuit Quantum circuit.
QuantumProgram Quantum Program Class.
QuantumRegister Implement a quantum register.
Reset Qubit reset.
Result Result Class.

Exceptions¶

QISKitError Base class for errors raised by the QISKit SDK.

Functions¶

available_backends Return names of backends that are available in the SDK, optionally filtering them based on their capabilities.
compile Compile a list of circuits into a qobj.
execute Executes a set of circuits.
get_backend Return an instance of a Backend object from its name identifier.
least_busy Return the least busy available backend for those that have a pending_jobs in their status.
load_qasm_file Construct a quantum circuit from a qasm representation (file).
load_qasm_string Construct a quantum circuit from a qasm representation (string).
local_backends Return the available local backends.
register Authenticate against an online backend provider.
registered_providers Return the currently registered providers.
remote_backends Return the available remote backends.
unregister Removes a provider from list of registered providers.
