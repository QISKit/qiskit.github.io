ベースクラス: logging.Formatter

Custom Formatter that uses a simple format for INFO.

Initialize the formatter with specified format strings.

Initialize the formatter either with the specified format string, or a default as described above. Allow for specialized date formatting with the optional datefmt argument (if omitted, you get the ISO8601 format).

Use a style parameter of '%', '{' or '$' to specify that you want to use one of %-formatting, str.format() ({}) formatting or string.Template formatting in your format string.

バージョン 3.2 で変更: Added the style parameter.

qiskit._logging.set_qiskit_logger()[ソース]¶

Update 'qiskit' logger configuration using a SDK default one.

Update the configuration of the 'qiskit' logger using the default SDK configuration provided by QISKIT_LOGGING_CONFIG:

  • console logging using a custom format for levels != INFO.
  • console logging with simple format for level INFO.
  • set logger level to INFO.

警告

This function modifies the configuration of the standard logging system for the 'qiskit.*' loggers, and might interfere with custom logger configurations.
