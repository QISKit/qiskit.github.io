qiskit.simulators._qasm_cpp_simulator.parse_complex(output, key)[ソース]¶

Parse complex entries in C++ simulator output.

This function converts complex numbers in the C++ simulator output into python complex numbers. In JSON c++ output complex entries are formatted as:

z = [re(z), im(z)]
vec = [re(vec), im(vec)]
ket = {'00':[re(v[00]), im(v[00])], '01': etc...}
パラメータ:
  • output (dict) – simulator output.
  • key (str) – the output key to search for.
