qiskit.qobj._converter.qobj_to_dict(qobj, version='1.0.0')[ソース]¶

Convert a Qobj to another version of the schema. Convert all types to native python types.

パラメータ:
  • qobj (Qobj) -- input Qobj.
  • version (string) -- target version for conversion.
戻り値:

dictionary representing the qobj for the specified schema version.
戻り値の型:

dict

QISKitError -- if the target version is not supported.
