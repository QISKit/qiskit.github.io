qiskit.transpiler.PassManager¶

class qiskit.transpiler.PassManager[ソース]¶

ベースクラス: object

PassManager class for the transpiler.

A PassManager instance is responsible for launching the requisite analysis and transformation passes on the quantum circuit, and to do this correctly & efficiently (i.e. keep track of dependencies between passes)

Methods

__init__() Initialize an empty PassManager object (with no passes scheduled).
add_pass(pass_) Schedule a pass in the passmanager.
passes() Return list of passes scheduled.
