qiskit.providers.ibmq.credentials._configrc module¶

Utilities for reading and writing credentials from and to configuration files.

Members¶

Functions¶

read_credentials_from_qiskitrc Read a configuration file and return a dict with its sections.
remove_credentials Remove credentials from qiskitrc.
store_credentials Store the credentials for a single account in the configuration file.
write_qiskit_rc Write credentials to the configuration file.

Reference¶

qiskit.providers.ibmq.credentials._configrc.read_credentials_from_qiskitrc(filename=None)[ソース]¶

Read a configuration file and return a dict with its sections.

パラメータ:filename (str) -- full path to the qiskitrc file. If None, the default location is used (HOME/.qiskit/qiskitrc).
戻り値:
dictionary with the contents of the configuration file, with
the form:
{credential_unique_id: Credentials}
戻り値の型:dict
Raises:QiskitError -- if the file was not parseable. Please note that this exception is not raised if the file does not exist (instead, an empty dict is returned).
qiskit.providers.ibmq.credentials._configrc.remove_credentials(credentials, filename=None)[ソース]¶

Remove credentials from qiskitrc.

パラメータ:
  • credentials (Credentials) -- credentials.
  • filename (str) -- full path to the qiskitrc file. If None, the default location is used (HOME/.qiskit/qiskitrc).
Raises:

QiskitError -- If there is no account with that name on the configuration file.
qiskit.providers.ibmq.credentials._configrc.store_credentials(credentials, overwrite=False, filename=None)[ソース]¶

Store the credentials for a single account in the configuration file.

パラメータ:
  • credentials (Credentials) -- credentials instance.
  • overwrite (bool) -- overwrite existing credentials.
  • filename (str) -- full path to the qiskitrc file. If None, the default location is used (HOME/.qiskit/qiskitrc).
Raises:

QiskitError -- if the account_name could not be assigned.
qiskit.providers.ibmq.credentials._configrc.write_qiskit_rc(credentials, filename=None)[ソース]¶

Write credentials to the configuration file.

パラメータ:
  • credentials (dict) --

    dictionary with the credentials, with the form:

    {credentials_unique_id: Credentials}
  • filename (str) -- full path to the qiskitrc file. If None, the default location is used (HOME/.qiskit/qiskitrc).
