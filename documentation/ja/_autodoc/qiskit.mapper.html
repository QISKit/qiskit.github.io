qiskit.mapper package¶

Utils for mapping.

Submodules¶

Exceptions¶

CouplingError Base class for errors raised by the coupling graph object.
MapperError Base class for errors raised by mapper module.

Classes¶

CouplingMap Directed graph specifying fixed coupling.
Layout Two-ways dict to represent a Layout.

Functions¶

euler_angles_1q Compute Euler angles for a single-qubit gate.
swap_mapper Map a DAGCircuit onto a CouplingGraph using swap gates.
two_qubit_kak Decompose a two-qubit gate over CNOT + SU(2) using the KAK decomposition.
