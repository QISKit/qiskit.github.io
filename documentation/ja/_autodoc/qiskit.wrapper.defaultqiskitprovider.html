available_backends(filters=None)[ソース]¶

Get a list of available backends from all providers (after filtering).

注釈

If two or more providers share similar backend names, only the backends belonging to the first registered provider will be returned.

パラメータ:filters (dict or callable) --

filtering conditions. each will either pass through, or be filtered out:

  1. dict: {'criteria': value}
    the criteria can be over backend's configuration or status e.g. {'local': False, 'simulator': False, 'operational': True}
  2. callable: BaseBackend -> bool
    e.g. lambda x: x.configuration['n_qubits'] > 5
戻り値:
a list of backend instances available
from all the providers.
戻り値の型:list[BaseBackend]
Raises:QISKitError -- if passing filters that is neither dict nor callable
