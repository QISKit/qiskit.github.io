qiskit.wrapper.least_busy¶

qiskit.wrapper.least_busy(names)[Quellcode]¶

Return the least busy available backend for those that have a pending_jobs in their status. Backends such as local backends that do not have this are not considered.

Parameter:names (list[str]) – backend names to choose from (e.g. output of available_backends())
Rückgabe:the name of the least busy backend
Rückgabetyp:str
Raises:QISKitError – if passing a list of backend names that is either empty or none have attribute pending_jobs
