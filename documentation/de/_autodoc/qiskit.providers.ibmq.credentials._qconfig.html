qiskit.providers.ibmq.credentials._qconfig module¶

Utilities for reading credentials from the deprecated Qconfig.py file.

Members¶

Functions¶

read_credentials_from_qconfig Read a QConfig.py file and return its credentials.

Reference¶

qiskit.providers.ibmq.credentials._qconfig.read_credentials_from_qconfig()[Quellcode]¶

Read a QConfig.py file and return its credentials.

Rückgabe:dictionary with the credentials, in the form:
{credentials_unique_id: Credentials}
Rückgabetyp:dict
Raises:QiskitError – if the Qconfig.py was not parseable. Please note that this exception is not raised if the file does not exist (instead, an empty dict is returned).
