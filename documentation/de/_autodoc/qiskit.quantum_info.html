qiskit.quantum_info package¶

Quantum Information methods.

Subpackages¶

  • qiskit.quantum_info.analyzation package
  • qiskit.quantum_info.operators package
  • qiskit.quantum_info.states package

Classes¶

Pauli A simple class representing Pauli Operators.

Functions¶

basis_state Return a basis state ndarray.
pauli_group Return the Pauli group with 4^n elements.
process_fidelity Return the process fidelity between two quantum channels.
projector maps a pure state to a state matrix
random_state Return a random quantum state from the uniform (Haar) measure on state space.
state_fidelity Return the state fidelity between two quantum states.
