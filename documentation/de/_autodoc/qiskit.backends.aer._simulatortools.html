qiskit.backends.aer._simulatortools.einsum_matmul_index(gate_indices, number_of_qubits)[Quellcode]¶

Return the index string for Numpy.eignsum matrix multiplication.

The returned indices are to perform a matrix multiplication A.B where the matrix A is an M-qubit matrix, matrix B is an N-qubit matrix, and M <= N, and identity matrices are implied on the subsystems where A has no support on B.

Parameter:
  • gate_indices (list[int]) – the indices of the right matrix subsystems to contract with the left matrix.
  • number_of_qubits (int) – the total number of qubits for the right matrix.
Rückgabe:

An indices string for the Numpy.einsum function.
Rückgabetyp:

str
Raises:
  • QISKitError – if the total number of qubits plus the number of
  • contracted indices is greater than 26.

qiskit.backends.aer._simulatortools.regulate(theta)[Quellcode]¶
Return the regulated symbolic representation of theta::
  • if it has a representation close enough to pi transformations,
    return that representation (for example, 3.14 -> sympy.pi).
  • otherwise, return a sympified representation of theta (for example,
    1.23 -> sympy.Float(1.23)).

See also UGateGeneric.

Parameter:theta (float or sympy.Basic) – the float value (e.g., 3.14) or the symbolic value (e.g., pi)
Rückgabe:the sympy-regulated representation of theta
Rückgabetyp:sympy.Basic
