Compute the mean value of an diagonal observable.

Takes in an observable in dictionary format and then calculates the sum_i value(i) P(i) where value(i) is the value of the observable for state i.

  • on qubits equals {"00" (ZZ) – 1, „11“: 1, „01“: -1, „10“: -1}

get_data(circuit=None)[Quellcode]¶

Get the data of circuit name.

The data format will depend on the backend. For a real device it will be for the form:

"counts": {'00000': XXXX, '00001': XXXX},
"time"  : xx.xxxxxxxx

for the qasm simulators of 1 shot:

'statevector': array([ XXX,  ..., XXX]),
'classical_state': 0

for the qasm simulators of n shots:

'counts': {'0000': XXXX, '1001': XXXX}

for the unitary simulators:

'unitary': np.array([[ XX + XXj
                       ...
                       XX + XX]
                     ...
                     [ XX + XXj
                       ...
                       XX + XXj]]
Parameter:circuit (str or QuantumCircuit or None) – reference to a quantum circuit If None and there is only one circuit available, returns that one.
Rückgabe:A dictionary of data for the different backends.
Rückgabetyp:dict
Raises:QISKitError – if there is no data for the circuit, or an unhandled error occurred while fetching the data.

get_qubitpol_vs_xval(nqubits, xvals_dict=None)[Quellcode]¶

Compute the polarization of each qubit for all circuits.

Compute the polarization of each qubit for all circuits and pull out each circuits xval into an array. Assumes that each circuit has the same number of qubits and that all qubits are measured.

Parameter:
  • nqubits (int) – number of qubits
  • xvals_dict (dict) – xvals for each circuit {‚circuitname1‘: xval1,…}. If this
  • none then the xvals list is just left as an array of zeros (is) –
Rückgabe:

mxn double array where m is the number of circuit, n the number of qubits xvals: mx1 array of the circuit xvals
Rückgabetyp:

qubit_pol
