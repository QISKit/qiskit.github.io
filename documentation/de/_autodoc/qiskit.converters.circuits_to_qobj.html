qiskit.converters.circuits_to_qobj.circuits_to_qobj(circuits, backend_name, config=None, shots=1024, max_credits=10, qobj_id=None, basis_gates=None, coupling_map=None, seed=None, memory=False)[Quellcode]¶

Convert a list of circuits into a qobj.

Parameter:
  • circuits (list[QuantumCircuits] or QuantumCircuit) – circuits to compile
  • backend_name (str) – name of runner backend
  • config (dict) – dictionary of parameters (e.g. noise) used by runner
  • shots (int) – number of repetitions of each circuit, for sampling
  • max_credits (int) – maximum credits to use
  • qobj_id (int) – identifier for the generated qobj
  • basis_gates (list[str])) – basis gates for the experiment
  • coupling_map (list) – coupling map (perhaps custom) to target in mapping
  • seed (int) – random seed for simulators
  • memory (bool) – if True, per-shot measurement bitstrings are returned as well
Rückgabe:

the Qobj to be run on the backends
Rückgabetyp:

Qobj
