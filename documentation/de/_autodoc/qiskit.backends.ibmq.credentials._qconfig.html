qiskit.backends.ibmq.credentials._qconfig.read_credentials_from_qconfig()[Quellcode]¶

Read a QConfig.py file and return its credentials.

Rückgabe:dictionary with the credentials, in the form:
{credentials_unique_id: Credentials}
Rückgabetyp:dict
Raises:QISKitError – if the Qconfig.py was not parseable. Please note that this exception is not raised if the file does not exist (instead, an empty dict is returned).
