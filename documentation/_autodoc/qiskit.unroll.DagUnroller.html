expand_gates(basis=None)[source]¶

Expand all gate nodes to the given basis.

If basis is empty, each custom gate node is replaced by its implementation over U and CX. If basis contains names, then those custom gates are not expanded. For example, if “u3” is in basis, then the gate “u3” will not be expanded wherever it occurs.

This member function replicates the behavior of the unroller module without using the OpenQASM parser.
