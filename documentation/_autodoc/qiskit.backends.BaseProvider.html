aliased_backend_names()[source]¶

Returns dict that defines alias names, usually shorter names for referring to the backends.

If an alias key is used, the corresponding backend will be chosen in order of priority from the value list, depending on availability.

Returns:list[str]]: {alias_name: list(backend_name)}

deprecated_backend_names()[source]¶

Dict that stores the current name for all deprecated backends. The conversion to the current name is done with a warning. These can be gradually removed in subsequent releases.
