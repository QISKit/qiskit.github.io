qiskit.wrapper package¶

Helper module for simplified QISKit usage.

The functions in this module provide convenience helpers for accessing commonly used features of the SDK in a simplified way. They support a small subset of scenarios and flows: for more advanced usage, it is encouraged to instead refer to the documentation of each component and use them separately.

Submodules¶

defaultqiskitprovider Meta-provider that aggregates several providers.

Functions¶

available_backends Return the backends that are available in the SDK, optionally filtering them based on their capabilities.
compile Compile a list of circuits into a qobj.
execute Executes a set of circuits.
get_backend Return an instance of a Backend object from its name identifier.
local_backends Return the available local backends.
register Authenticate against an online backend provider.
remote_backends Return the available remote backends.
