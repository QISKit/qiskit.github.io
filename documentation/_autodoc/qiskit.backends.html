qiskit.backends package¶

Utilities for using backends.

Submodules¶

_qiskit_cpp_simulator Interface to C++ quantum circuit simulator with realistic noise.
_qasmsimulator Contains a (slow) python simulator.
_unitarysimulator Contains a (slow) Python simulator that returns the unitary of the circuit.
_projectq_simulator Backend for the Project Q C++ simulator.

Classes¶

BaseBackend Base class for backends.

Functions¶

discover_local_backends This function attempts to discover all backend modules.
discover_remote_backends Discover backends available on the Quantum Experience
get_backend_class Return the class object for the named backend.
get_backend_configuration Return the configuration for the named backend.
get_backend_instance Return a backend instance for the named backend.
local_backends Get the local backends.
register_backend Register a backend in the list of available backends.
remote_backends Get the remote backends.
update_backends Update registered backends.
