qiskit.mapper.coupling_list2dict¶

qiskit.mapper.coupling_list2dict(couplinglist)[source]¶

Convert coupling map list into dictionary.

Example list format: [[0, 1], [0, 2], [1, 2]] Example dictionary format: {0: [1, 2], 1: [2]}

We do not do any checking of the input.

Return coupling map in dict format.
