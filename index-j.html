QISKit

量子情報ソフトウェア開発キット

(Quantum Information Software Kit)

最新バージョン v0.3.6

量子情報ソフトウェア開発キットであるQISKitはIBM Q experience (QX)とOpenQASMを利用するためのソフトウェア開発キットです．

利用方法

QISKitで量子コンピューターのプログラムを記述して，コンパイルし，オンラインで稼働している実デバイスやシミュレーター等のバックエンドで実行．

チュートリアル

マニュアル

量子プログラムの実行

[python3] $ pip install qiskit


  from qiskit import QuantumProgram
  qp = QuantumProgram()
  qr = qp.create_quantum_register('qr', 2)
  cr = qp.create_classical_register('cr', 2)
  qc = qp.create_circuit('Bell', [qr], [cr])
  qc.h(qr[0])
  qc.cx(qr[0], qr[1])
  qc.measure(qr[0], cr[0])
  qc.measure(qr[1], cr[1])
  result = qp.execute('Bell')
  print(result.get_counts('Bell'))

QISKitのインストール 

$ pip install qiskit
